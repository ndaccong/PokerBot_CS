# Poker bot

Are you a poker fan? If the answer is yes, let's try beating this bot. This is a simple bot yet quite difficult to win against. Just follow these steps to run the game and experience it yourself.

1. [Install Splashkit](https://splashkit.io/installation/)
2. In the terminal, from the main directory, run this command:
`skm dotnet run`

Enjoy!
